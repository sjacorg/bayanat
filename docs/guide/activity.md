# Activity Monitor

The Activity Monitor dashboard provides a list of all actions by all users in the system.

::: tip Admin only
Only administrators can access the Activity Monitor.
:::

## Log Entry Content

Each entry contains:

- Time of the action
- User who carried it out
- Type of the action
- Subject of the action (user, Actor, Bulletin, etc.)

## Tracked Actions

- Login and logout
- Item created
- Item updated
- Bulk operations

Items tracked: users, Actors, Bulletins, and Incidents.

::: info
The Activity Log does not currently show the actual content of changes. For that, see [Revision History](/guide/revision-history).
:::
