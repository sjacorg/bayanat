# Locations

Locations is a central list of entries for administrative centers (cities, towns, villages, etc.). Bayanat currently follows the Syrian administrative system, but the model can accommodate other countries.

::: tip Admin only
Locations administration dashboard is available to admins only.
:::

## Fields

### Title

Bilingual location name.

### Latitude and Longitude

Coordinates of the location.

### Location Type

Governorate, district, subdistrict, or center.

### Parent

Direct parent of the location.

### Parent (G) / Parent (D) / Parent (S) / Parent (C)

Specific parent levels: Governorate, District, Subdistrict, Center.

## Import from CSV

Admins can bulk import locations from a formatted CSV file.
