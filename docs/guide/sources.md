# Sources

Sources are profiles of the original creators and publishers of the content of Actors and Bulletins. This structure provides easy search and filtering and prevents data entry mistakes.

::: tip Admin only
Sources administration dashboard is available to admins only.
:::

## Fields

### Title

Name of the source.

### Parent

Creates a hierarchy of parent and child Sources. Useful for search.

### ETL ID

Read-only field for ETL operations. Can hold a unique identifier (e.g., an uploader user ID on a video platform).

### Comments

Additional information about the source (website, contact details, etc.).

## Import from CSV

Admins can bulk import sources from a formatted CSV file.
