# Events

Events are data forms to store time and location information related to Actors, Bulletins, and Incidents. Users can attach one or more events.

## Event Fields

### Title

Bilingual short description of the event.

### Comment

Bilingual field for additional context (e.g., the source of the information).

### Location

The location of the event.

### Event Type

Select from a list of pre-defined event types.

### From and To

Two time fields. Use "from" for a specific time, and both "from" and "to" for a period.

### Estimated Time

Checkbox to indicate that the provided time is not precise.

## Event Types

Admins can add and modify event types. The visibility of event types in Actors and Bulletins can be controlled.

::: tip Admin only
Event types dashboard is only visible to administrators.
:::

### Event Type Fields

- **Title**: Bilingual name visible to all users
- **Comment**: Bilingual admin note
- **Visibility controls**: Checkboxes to control visibility in Actors and Bulletins

### Import from CSV

Admins can bulk import event types from a formatted CSV file.
